# Componente Practico Programacion - Software FJ

Este proyecto consiste en el desarrollo de un sistema en Python para gestionar clientes, servicios y reservas. Fue realizado como parte del curso de Programación.

El sistema permite:
- Registrar clientes
- Crear diferentes tipos de servicios (salas, equipos y asesorías)
- Realizar reservas
- Manejar errores sin que el programa se detenga
- Guardar errores en un archivo de logs

Y en el desarrollo se aplicaron:
- Abstracción
- Herencia
- Polimorfismo
- Encapsulación
- Manejo de excepciones (try, except, else, finally)

# Tecnologías utilizadas
Se utilizaron tecnologias como: 
- Python 3
- Programación Orientada a Objetos (POO)
- Manejo de archivos
- Git y GitHub
